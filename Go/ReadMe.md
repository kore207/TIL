### Go lang study

> Go는 전통적인 컴파일, 링크 모델을 따르는 범용 프로그래밍 언어이다.
> 
> C++, Java, Python의 장점들을 뽑아 만들어 졌다. 
> 
> C++와 같이 Go는 컴파일러를 통해 컴파일되며, 정적 타입의 언어이다. 
> 
> Garbage Collection 기능을 제공한다.
